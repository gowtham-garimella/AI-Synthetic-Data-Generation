# AI-Synthetic-Data-Generation

# 🧠 AI Synthetic Data Generation & Analysis

## 📌 Project Overview
This project generates synthetic data using **CTGAN (Generative AI)** and evaluates its quality using machine learning models.

It compares original vs synthetic data and checks if synthetic data can be used reliably for training ML models.

---

## ⚙️ Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib
- SDV (CTGAN)

---

## 📊 Features
- Data preprocessing and cleaning
- Label encoding
- Synthetic data generation using CTGAN
- Anomaly detection using Isolation Forest
- Model training using Random Forest
- Performance evaluation (Accuracy, MAE, MSE)
- ROC Curve visualization

---

## 📁 Dataset
- Adult Income Dataset (UCI Repository)

---

## 🚀 How to Run

```bash
pip install -r requirements.txt
python src/main.py
```

---

## 📈 Results

- High similarity between original and synthetic data
- Model accuracy > 80%
- Synthetic data preserves real-world patterns

---

## 🧪 Output Highlights

- Synthetic dataset generated
- ROC Curve plotted
- Accuracy & Loss graphs
- Anomaly comparison between datasets

---

## 📌 Conclusion

Synthetic data generated using CTGAN is:
- Realistic
- Diverse
- Suitable for machine learning tasks
